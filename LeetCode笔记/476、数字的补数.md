# 476、数字的补数

## 题目：

给定一个正整数，输出它的补数。补数是对该数的二进制表示取反。

**示例 1:**

```
输入: 5
输出: 2
解释: 5 的二进制表示为 101（没有前导零位），其补数为 010。所以你需要输出 2 。
```

**示例 2:**

```
输入: 1
输出: 0
解释: 1 的二进制表示为 1（没有前导零位），其补数为 0。所以你需要输出 0 。
```

 

**注意:**

1. 给定的整数保证在 32 位带符号整数的范围内。
2. 你可以假定二进制数不包含前导零位。
3. 本题与 1009 相同

## 解答：

```csharp
public static int FindComplement(int num)
{

    // 111 ->1000-1=111
    //-101
    // 010 ^方法或者相减    
    //根据题意，只要和1111相减就可以了
    //long temp = 1;

    //    //0010 - 2
    //    //0100 - 4
    //    temp <<= 1;
    //}

    long temp = 1;
    while (temp <= num)
    {
        temp <<= 1;
    }
    return (int)((temp - 1) ^ num);
}
```
